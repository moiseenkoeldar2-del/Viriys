# Cybersecurity Matrix Game

Интерактивный веб-квест на тему информационной безопасности и обхода систем блокировки.

## Описание логики
* Всего доступно **3 попытки** для ввода правильного секретного ключа дешифрования.
* После **2-й неудачной попытки** терминал разворачивается на весь экран и активирует защитную блокировку на **1 час (01:00:00)**.
* Если последняя **3-я попытка** оказывается неверной, включается аварийный режим тревоги.
